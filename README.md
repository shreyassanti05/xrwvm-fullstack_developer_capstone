# Cars Dealership Full Stack Capstone Project

This project is a full-stack web application developed as part of the IBM Full Stack Software Developer Capstone. The application simulates a car dealership platform where users can browse dealership locations, view dealer information, read customer reviews, and submit reviews after logging in.

The frontend is built using React to provide a responsive and user-friendly interface, while the backend uses Django, Flask, and Node.js microservices to handle authentication, dealership data, review management, and sentiment analysis. MongoDB and SQLite are used for data storage, and REST APIs enable communication between different services.

The application includes user registration and authentication, dealership search and filtering, dealer detail pages, review submission, sentiment analysis of reviews, and an administrative interface for managing application data. The project also demonstrates modern software development practices, including Git and GitHub for version control, GitHub Actions for CI/CD, Docker and Kubernetes for containerization and deployment, and IBM Cloud Code Engine for cloud hosting.

This capstone showcases practical skills in full-stack web development, RESTful API integration, microservices architecture, cloud deployment, responsive UI development, DevOps, and collaborative software engineering.
