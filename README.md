# fullstack_developer_capstone

A national car retailer in the U.S. web application that displays dealership branches and allows users to view and submit reviews.

## Technologies Used
- Frontend: React, HTML, CSS
- Backend: Django, Flask, Node.js
- Database: MongoDB, SQLite
- Deployment: Docker, Kubernetes, IBM Cloud Code Engine

## Features
- View all dealership branches.
- Filter dealerships by state.
- View customer reviews for specific dealerships.
- Submit new reviews with sentiment analysis integration.
- Admin dashboard for managing reviews and dealers.
